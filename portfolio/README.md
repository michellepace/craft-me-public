# Portfolio

## INDEX.xml

Searchable index of all portfolio entries. Each `<entry>` contains metadata and a 30-50 word summary optimised for semantic search by hiring managers.

## Subdirectories

- `website/` - Articles and project write-ups from [ailearnlog.com](https://ailearnlog.com). Cleaned markdown versions of published posts.

- `repos/` - Technical deep-dives on individual repositories, documenting architecture, developer workflow, and PM-relevant signals.
